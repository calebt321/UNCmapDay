Sentiments of Fishermen Towards NC Fishieries Managment

Caleb Tyler

The following map uses data about commercial and recreational fishermens interactions with state fisheries managment. The data was collected via mail-in and online surveys by researchers from ECU and UNC's Geography department. Some key differences in patterns between commercial and recreation fishermen were collected and displayed in the below maps. A more comprehensive map can be viewed using the web map viewer after logging in with your ONYEN.
